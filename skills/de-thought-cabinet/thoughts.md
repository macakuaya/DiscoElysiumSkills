### Clear Enough For Cuno
- **Internalized**: 2026-02-28
- **Problem**: Diagrams, explanations, and documentation default to mirroring the system's internal structure — code state names, implementation flow, jargon — producing output that's accurate to the machine but illegible to the human
- **Completion Effect**: Before producing any visual or written explanation meant for others, ask: "Would someone with zero context understand this in 10 seconds?"
